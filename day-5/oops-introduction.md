# OBJECT ORIENTED PROGRAMMING (OOP)

## What is OOP?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code using objects and classes.

Instead of focusing only on functions, OOP focuses on modeling real-world entities.

Examples:

* Student
* Employee
* Car
* Bank Account
* User

## Why OOP?

Benefits:

* Code Reusability
* Better Organization
* Scalability
* Maintainability
* Security

Large applications use OOP extensively.

Examples:

* Django
* Flask
* FastAPI
* React Components
* Java Applications

## Real World Example

Think of a Car.

Car has:

Properties:

* Brand
* Model
* Color

Behaviors:

* Start
* Stop
* Accelerate

In OOP:

Properties → Attributes

Behaviors → Methods

## Class vs Object

Class: Blueprint.

Object: Real instance created from blueprint.

Example:

Class: Car

Objects:
BMW
Audi
Tesla

## Student Example

Class: Student

Objects:
Alex
John
Emma

All are students but have different values.

## Interview Questions

### Q1. What is a Class?

Answer: A blueprint used to create objects.

### Q2. What is an Object?

Answer: An instance of a class.

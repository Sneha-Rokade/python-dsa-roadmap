# INTRODUCTION TO PYTHON

## What is Python?

Python is a high-level, interpreted, general-purpose programming language created by Guido van Rossum and first released in 1991.

### High-Level Language

Python is human-readable and easy to understand.

Example:

name = "Sneha"
print(name)

Compared to languages like C, Python requires less code and is easier to read.

### Interpreted Language

Python executes code line by line using an interpreter.

Execution Flow:

Python Code
     ↓
Python Interpreter
     ↓
Output

Unlike C or C++, Python does not require a separate compilation step before execution.

## Why is Python So Popular?

### 1. Less Code

Python allows developers to accomplish tasks with fewer lines of code.

Example: Swapping Two Numbers

Python:

a, b = b, a

Traditional Approach:

temp = a
a = b
b = temp

This increases developer productivity.

### 2. Easy Syntax

Python syntax is simple and resembles English.

Example:

if age >= 18:
    print("Adult")

Benefits:

* Easy to learn
* Easy to read
* Easy to maintain

### 3. Fast Development

Companies can build products quickly using Python.

Examples:

* Startup MVPs
* Internal Tools
* Automation Scripts
* APIs

Python helps reduce development time and cost.

### 4. Huge Ecosystem and Libraries

Python has thousands of ready-to-use libraries.

Examples:

| Purpose             | Library      |
| ------------------- | ------------ |
| Data Analysis       | Pandas       |
| Numerical Computing | NumPy        |
| Machine Learning    | Scikit-Learn |
| Deep Learning       | TensorFlow   |
| Deep Learning       | PyTorch      |
| Backend APIs        | FastAPI      |
| Web Development     | Django       |
| Automation          | Selenium     |
| Data Visualization  | Matplotlib   |

Developers can solve complex problems without building everything from scratch.

## Interview Questions

### Q1. What is Python?

Python is a high-level, interpreted, general-purpose programming language used for web development, automation, AI/ML, cloud computing, and many other domains.

### Q2. Why is Python called an interpreted language?

Because Python code is executed by an interpreter line by line without requiring a separate compilation step.

### Q3. What makes Python different from Java or C++?

* Simpler syntax
* Less code
* Faster development
* Dynamic typing
* Large ecosystem of libraries

### Q4. Why is Python popular in AI and Machine Learning?

Because it provides powerful libraries such as NumPy, Pandas, TensorFlow, PyTorch, and Scikit-Learn that simplify AI/ML development.

* Web Development
* Automation
* AI/ML

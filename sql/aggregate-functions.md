================================================
AGGREGATE FUNCTIONS
================================================

Aggregate functions operate
on multiple rows and return
a single value.

================================================
COUNT()
================================================

Count rows.

SELECT COUNT(*)
FROM employees;

================================================
SUM()
================================================

Total salary.

SELECT SUM(salary)
FROM employees;

================================================
AVG()
================================================

Average salary.

SELECT AVG(salary)
FROM employees;

================================================
MAX()
================================================

Highest salary.

SELECT MAX(salary)
FROM employees;

================================================
MIN()
================================================

Lowest salary.

SELECT MIN(salary)
FROM employees;

================================================
INTERVIEW QUESTION
================================================

Q. What are aggregate functions?

Answer: Functions that operate on multiple rows
and return a single result.

## What is SQL?

SQL stands for:

Structured Query Language
SQL is a language used to communicate with databases.

Using SQL we can:
- Store Data
- Retrieve Data
- Update Data
- Delete Data

SQL is used in almost every software application.

Examples:

- Banking Systems
- E-Commerce Applications
- Social Media Platforms
- Hospital Management Systems
- Employee Management Systems

------------------------------------------------

## Why Do We Need SQL?

Imagine storing employee information.

Employee 1:
Name: Alex
Age: 25
Role: Engineer

Employee 2:
Name: John
Age: 30
Role: Manager

If we store thousands of employees in files,
searching becomes difficult.

Databases help us:

- Store large amounts of data
- Search data quickly
- Update data easily
- Maintain security

SQL is used to interact with these databases.

------------------------------------------------

## What is DBMS?

DBMS stands for:

Database Management System

A DBMS is software used to create and manage databases.

Examples:

- MySQL
- PostgreSQL
- Oracle
- SQL Server

Functions:

- Store Data
- Retrieve Data
- Update Data
- Delete Data

------------------------------------------------

## What is RDBMS?

RDBMS stands for:

Relational Database Management System
Data is stored in tables.
Tables can be connected using relationships.

Example:

Employees Table

| EmployeeID | Name |
|------------|------|
| 1          | Alex |

Departments Table

| DepartmentID | EmployeeID |
|--------------|------------|
| 101          | 1          |

EmployeeID connects both tables.

------------------------------------------------

## DBMS vs RDBMS

DBMS:

- Stores data
- May not support relationships

RDBMS:

- Stores data in tables
- Supports relationships
- Uses Primary Keys and Foreign Keys

Examples:

DBMS:
- File System

RDBMS:
- PostgreSQL
- MySQL
- Oracle
